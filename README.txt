АвтоВердикт — сборка сайта
================================

Как развернуть
--------------
1) Залейте все файлы на хостинг (Apache/Nginx+PHP).
2) Убедитесь, что `send.php` доступен по адресу `https://ВАШ_ДОМЕН/send.php`.
3) В `index.html` поле `FORM_ENDPOINT` уже указано как 'send.php', а `DEMO_MODE` автоматически
   включается только если страницу открывают локально (`file://`). На сервере форма отправит POST JSON на send.php.

Тестирование
------------
- Откройте `index.html` локально — форма будет работать в DEMO-режиме (без отправки, просто покажет успех).
- На сервере: заполните форму, письмо придёт на `auto.verdict24@gmail.com`. 
  Если письма не приходят — проверьте логи почты, SPF/DKIM или используйте SMTP-отправку через почтовый сервис.

Замена изображений
------------------
- Логотип: `assets/logo-avtoverdikt.png`
- Сертификаты: `assets/cert-1.jpg` ... `assets/cert-4.jpg`
- Фоновое фото в hero задаётся в стиле `.hero` (внутри index.html) — замените URL на свой.

Лицензии и библиотеки
---------------------
